SimpleKernel
============

A simple hobby kernel that doesn't do much

PREREQUISITES:
-------------
1. GCC (tested with 5.2.0)
2. NASM (tested with 2.11)
3. QEMU (tested with 2.4.0)

TO BUILD:
-------------
>
>$ make<br />

TO CLEAN:
-------------
>
>$ make clean<br />

TO TEST (after building):
-------------
>
>$ cd ./out<br />
>$ ./run_qemu.sh<br />
