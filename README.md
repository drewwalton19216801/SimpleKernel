SimpleKernel
============

A simple hobby kernel that doesn't do much

PREREQUISITES:
-------------
1. GCC (tested with 4.8.1)
2. NASM (tested with 2.10.09)
3. QEMU (tested with 1.5.2)

TO BUILD:
-------------
>
>$ cd src/<br />
>$ make<br />

TO TEST (after building):
-------------
>
>$ cd ../out/<br />
>$ ./run_qemu.sh<br />
